01 — Brief

Making Data Science Human

Data Reply — part of the international Reply Group — signed an exclusive agreement with The Data Incubator, a leading programme dedicated to training professional Data Scientists. The brief: create a series of infographics explaining the path to becoming a data scientist, making complex concepts around Big Data, Machine Learning, and advanced analytics accessible to a broad professional audience.

Data science is abstract by nature — algorithms, models, pipelines, and statistical methods that exist in code and mathematics, not in physical form. The challenge was to translate this invisible world into a clear, engaging visual narrative that could work across digital channels, presentations, and training materials.

02 — The System

Five Infographics, One Visual Language

The series was designed as a cohesive system — a consistent visual language using flat illustration, structured grids, and a colour palette that communicates both the precision of data work and the human element of learning and growth.

The icons and illustration style were developed from scratch to represent abstract technological concepts — cloud computing, machine learning models, data pipelines — in a form immediately readable without technical background knowledge. Each infographic addresses a different dimension of the data scientist's journey: the technologies involved, the skills required, the tools of the trade, and the career path from analyst to scientist.
